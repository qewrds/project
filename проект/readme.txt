Проект PyQt5 «Приложение для работы с кофейными автоматами»

  1. Проект.

 Приложение-помощник для отслеживания кол-ва стаканчиков кофе, которые может приготовить автомат.

  Приложение имеет следующие функции:

    1)Для каждого из автоматов подсчитывать оставшиеся запасы кофе каждого из видов, в случае необходимости сообщать о том, что запасы нужно пополнить.
    2)Хранить и показывать список неисправностей автоматов, которые требуется устранить, для каждой из неисправностей должна быть возможность добавить изображение.

  2. Работа с приложением.

 Основная работа с приложением производится через главное окно, имеющее две вкладки — для запасов и поломок.

    1)Запасы:
      На вкладке с запасами можно контролировать оставшиеся в каждом из автоматов кофе (запасы, подходящие к концу, подсвечиваются красным);
    Также на этой вкладке можно вводить в программу информацию о добавлении и расходе .

    2)Поломки:
      На вкладке с поломками отображаются все зарегистрированные в системе и ещё не устранённые поломки.
    Дважды кликнув на поломку, можно просмотреть подробную информацию о ней, и, после устранения, уведомить систему о том, что поломка устранена.
    Также на этой вкладке вводится информация о возникших поломках.

  3. Меню.

     Меню в верхней части экрана приложения позволяет редактировать информацию о местах, где находятся автоматы.
